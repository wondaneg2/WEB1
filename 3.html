JavaScript (/'d3a;va,skript/),[6]often abbreviated as JS, is a high-level, interpreted programming language. Alongside HTML and CSS, JavaScript is one of the three core technologies of the World Wide Web.[7]JavaScript enables interactive web pages and thus is an essential part of web applications.
